Sobre Mí

Hola, soy Larry Baltodano. Soy un desarrollador Backend en formación y estudiante de Ingeniería en Sistemas, con experiencia en Java y Spring Boot para la creación de APIs REST seguras. También manejo Python para backend con FastAPI y Flask, además de scripts automatizados. Aunque mi enfoque principal es Java, planeo explorar Django y php junto a Laravel en el futuro.

Actualmente, profundizo en patrones de diseño y arquitectura de microservicios, explorando Docker y Kubernetes. Todo con el fin de seguir aprendiendo y sobre todo ampliar mis conocimientos para aplicarlos dentro de un entorno laboral y hasta de transmitirlo a otros
